# BuildCloud Distributed Code Compilation Simulator

## Overview

BuildCloud is a terminal-based C++ project inspired by distributed build systems such as GitHub Actions and Jenkins.

The project demonstrates the practical application of Data Structures and Algorithms for compilation management, dependency tracking, scheduling, searching, sorting, shortest path calculation, and memory allocation.

## Features

### 1. Package Tracker
- Binary Search Tree (BST)
- Fast module insertion and search

### 2. Build History
- Stack implementation
- Undo previous build operations

### 3. Task Line
- Queue implementation
- FIFO compilation scheduling

### 4. File Finder
- Binary Search
- Fast file lookup

### 5. Change Sorter
- Merge Sort
- Sort modules by modification count

### 6. Dependency Map
- Graph representation
- BFS and DFS traversal

### 7. Fastest Build Path
- Dijkstra's Algorithm
- Minimum compilation path calculation

### 8. Memory Manager
- Greedy allocation strategy
- Efficient resource utilization
